Esta base de datos está diseñada para gestionar la información relacionada con estudiantes, docentes, materias, horarios y las asesorías asignadas. La tabla"Estudiante" almacena datos como número de control, nombre, carrera y semestre. La tabla "Docente" contiene datos personales y credenciales de los profesores. La tabla "Materia" define las materias mediante claves únicas, nombres completos y cortos. La tabla "Horario" organiza los días y horas disponibles para actividades académicas, relacionadas con los docentes. Finalmente, la tabla "Asesoría" vincula estudiantes, docentes y materias, detallando fechas y horarios específicos para cada asesoría. Las relaciones entre las tablas garantizan un manejo eficiente y estructurado de los datos.
